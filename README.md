# Sistema de Gestión de Pedidos para una Cafetería

## alum. Madelein Rodriguuez 



## Objetivo del sistema

Desarrollar un sistema que permita gestionar pedidos en una cafetería mediante un menú interactivo. El programa facilita la visualización del menú de productos, la selección de artículos y el cálculo automático del valor total del pedido.

## Funcionalidades

- Mostrar el menú de productos.
- Agregar productos al pedido.
- Calcular el total de la compra.
- Vaciar el pedido.
- Finalizar la ejecución del sistema.

## Ejecución

1. Abrir el notebook en Google Colab.
2. Ejecutar todas las celdas en orden.
3. Utilizar el menú interactivo para realizar pedidos.

## Estructuras de datos utilizadas

- Tuplas para almacenar las categorías del menú.
- Diccionario para relacionar cada producto con su precio.
- Lista para almacenar los productos seleccionados por el cliente.

## Tecnologías utilizadas

- Python
- Google Colab
